# Online Voting System

A secure web-based voting platform with a unique 
percentage-based voting mechanism.

## 🎯 Unique Feature
**Percentage-based voting** — voters can distribute 
their votes across multiple candidates 
(e.g., 20% to A, 50% to B, 30% to C)

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** Git, VS Code

## ✨ Features
- Secure voter authentication
- Admin and voter modules
- Candidate management
- Percentage-based vote distribution
- Real-time result visualization
- MySQL database integration

## 🚀 How to Run
```bash
# Clone the repo
git clone https://github.com/tusharautade27/voting-system

# Install dependencies
pip install -r requirements.txt

# Setup database
mysql -u root -p < schema.sql

# Run the app
python app.py
```
